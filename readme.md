#install 
1)  clone repository
2) $ cd to repository
3) $ composer install
4) $ npm install
5) $ cp .env.example .env
6)  set up db settings to .env
7) $ php artisan migrate
8) $ npm run dev

#using
run in 2 console: <br>
1) $ php artisan serve
2) $ php artisan websocket:init

Go to http://127.0.0.1:8000/ and enjoy:)
